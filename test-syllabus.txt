COURSE DESCRIPTION
------------------
This course provides an introduction to fundamental computer science concepts 
including programming, data structures, and algorithms. Students will learn 
to think computationally and solve problems using programming languages.

LEARNING OBJECTIVES
-------------------
Upon completion of this course, students will be able to:
1. Understand basic programming concepts and syntax
2. Implement common data structures (arrays, lists, stacks, queues)
3. Analyze algorithm complexity and efficiency
4. Write clean, maintainable code
5. Debug and test programs effectively
